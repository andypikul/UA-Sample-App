Class Overview

A web view client that intercepts Urban Airship URLs and enables triggering actions from javascript.

The UAWebViewClient will intercept links with the 'uairship' scheme and with the commands (supplied as the host) 'run-actions' or 'run-basic-actions'.

The run-actions command runs a set of actions listed in the URL's query options, by providing key=value pairs, where each pair's key is the name of an action and the value is a JSON encoded string representing the value of the action's ActionArguments.
